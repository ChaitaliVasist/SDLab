HeapSort

Heapsort is a comparison-based sorting algorithm that uses a binary heap data structure.
It is an in-place algorithm (requires no extra space apart from the input array) with a time complexity of O(n log n).

Travelling Sales Man

The Traveling Salesman Problem (TSP) is a classic optimization problem where the goal is:
To find the shortest possible route that visits each city exactly once and returns to the starting city.
It is represented as a graph, where cities are nodes and distances between them are edges.

Limitations of Algorithm Power

The P vs NP problem explores the limitations of algorithms in solving computational problems. P represents problems solvable in polynomial time, while NP includes problems whose solutions can be verified in polynomial time. NP-complete problems are the hardest in NP; solving one efficiently would mean all NP problems can be solved efficiently, but it's unknown if P=NP

BackTracking

Backtracking is a general algorithmic technique used to solve problems incrementally by exploring all possible solutions. It builds solutions step-by-step and abandons a solution ("backtracks") as soon as it determines that it cannot lead to a valid solution.
Learn more about this after playing the NQueens game.

Branch and Bound

Branch and Bound is an algorithmic technique used to solve optimization problems by systematically exploring the solution space. It divides the problem into smaller subproblems (branching) and eliminates subproblems that cannot lead to the optimal solution (bounding).
